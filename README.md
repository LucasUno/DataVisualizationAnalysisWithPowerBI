分析数据来源 AdvantureWorksDW2025.bak数据集
dashBoard文件：Lucas's Dashboard Demo.pbix 是通过PowerBI制作(主要是免费可本地保存)

excel_Automation.py是关于库存相关库龄(>90天)货物的库存成本报告数据的处理脚本
      python脚本的优势在于能灵活的处理相关时间的计算以及数据清洗操作的便捷；
      处理不同系统平台的数据时优先用python脚本处理一下对后续的处理更友好
数据源：仓库成本表，库存源数据表
1、读取相关数据源
2、获取时间数据计算库龄，并筛选出大于90天库龄的数据
3、关联表格数据
4、计算仓库存储成本
5、输出报告

Sales 为 销售经营分析项目 的
    1.SQL取数逻辑  核心业务数据(销售额、销售总成本、利润率、同比增长)，并通过产品、客户、地理区域不同维度去分析其核心指标
    2.数据库导出的查询结果宽表
    
Inventory 为 进销存与库存周转分析项目 的
    1.SQL取数逻辑  核心业务数据(期末库存金额、库存移动平均值、库存周转率、库存周转天数)
    2.数据库导出的查询结果宽表
